# Semantic Chatbot with Offline Calendar

## Overview
A Python-based AI chatbot that uses Sentence Transformers for semantic intent recognition. The chatbot understands user messages, detects intents, extracts event details, and stores reminders in an offline calendar.

## Features
- Semantic intent classification
- Greeting detection
- Availability detection
- Event recognition
- Natural language date and time extraction
- Offline reminder storage
- Duplicate event detection

## Technologies Used
- Python
- Sentence Transformers
- JSON
- Regular Expressions
- datetime
