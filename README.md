# Semantic Chatbot with Offline Calendar

## Overview
A Python-based AI chatbot that uses Sentence Transformers for semantic intent recognition. The chatbot detects user intents, extracts event details from natural language, and stores reminders in an offline calendar.

## Features
- Semantic intent classification
- Greeting and availability detection
- Natural language date and time extraction
- Offline reminder storage using JSON
- Duplicate event detection

## Technologies Used
- Python
- Sentence Transformers
- JSON
- Regular Expressions
- datetime
